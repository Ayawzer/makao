# Makao (Python/Flask)

Prosta implementacja gry karcianej Makao działająca w przeglądarce.

## Jak uruchomić?

Upewnij się, że jesteś w folderze projektu i wykonaj te dwa kroki w konsoli:

### 1. Zainstaluj wymagane biblioteki

(Robisz to tylko raz po pobraniu projektu)

```bash
pip install -r requirements.txt
```

### 2. Uruchom grę

```bash
python app.py
```

### 3. Graj

Gra odpala się na localhoscie na porcie 12125 (http://127.0.0.1:12125)
